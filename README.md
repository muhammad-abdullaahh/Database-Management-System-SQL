# Database Management System — SQL

A collection of SQL scripts written during my **Database Management Systems** course. Covers core SQL concepts through lab exercises, practice queries, and assigned tasks.

## Topics Covered

- DDL: `CREATE`, `ALTER`, `DROP`
- DML: `INSERT`, `UPDATE`, `DELETE`, `SELECT`
- Filtering & sorting: `WHERE`, `ORDER BY`, `GROUP BY`, `HAVING`
- Joins: `INNER JOIN`, `LEFT JOIN`, etc.
- Subqueries and aggregate functions

## Tools Used

- **SQL Server / SSMS** (Microsoft SQL Server Management Studio)
